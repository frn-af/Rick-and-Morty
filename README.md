# Rick and Morty app

This is a simple app that uses the Rick and Morty API to display information about the characters dan location.

## feature 

- ui frendly
- system / dark theme
- infinite scrolling
- assign location
- character details
- location list

## installation

1. Clone the repository
2. Run `pnpm install`
3. Run `pnpm start`
